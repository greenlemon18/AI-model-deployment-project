# AI-model-deployment-project
  This project is just a personal interest project and the main component of this model is an AI visual recognition model. Once trained, i will employ this model onto a development board and then conduct actual testing to rvaluate the performance of the deployed model on the board. This project will go through 4 stages. 
  The first stage is AI model training. There are already numerous image recognition models available, among which YOLOv5 stands out as a relatively mature one. I have decided to use its model for training. The training set for the model will be captured from real videos, and then frames will be extracted from the captured videos. The extracted frames will be labeled. The images in the training set primarily encompass three types: unobstructed static images, obstructed static images, and static images with both interference and obstruction. Following these are dynamic images of the same type.
