训练参数记录
采用迁移学习，使用现有YOLOv8模型

参数	        默认值	设置	  含义	          调大/调小的效果
lr0	          0.01	  0.007	  初始学习率	    防止过大，模型不收敛，防止过小，收敛太慢，建议从小开始测试
lrf	          0.01	  0.007	  最终学习率	    配合余弦退火策略动态降低学习率，最终学习率 = lr0 × lrf
cos_lr	      False	  True	  余弦学习率衰减  学习率按余弦曲线平滑下降，比线性衰减更稳定，通常提升 0.5-1% mAP
momentum	    0.937	  0.937	  动量参数	      加速收敛，抑制震荡。默认值适用于大多数场景
weight_decay	0.0005	0.0005	权重衰减        L2正则化，防止过拟合。不宜过大，否则模型学不到特征
mosaic	      1.0	    1.0	    马赛克增强概率	1.0 表示每轮都做。提升小目标检测能力
mixup	        0.0	    0.2	    混合增强概率	  两张图按 α 权重混合。提升泛化能力，但不宜过大，会模糊特征
copy_paste	  0.0    	0.05	  复制增强概率	  把目标随机粘贴到其他图上，对遮挡场景特别有效
degrees	      0.0	    5.0	    随机旋转角度	  模拟摄像头倾斜。小球是圆形，旋转影响不大，可适度使用
translate	    0.0	    0.15	  随机平移比例	  模拟目标不在画面中心。0.15 表示最多平移 15% 的画面宽高
scale	        0.5	    0.5	    随机缩放幅度	  模拟不同距离。0.5 表示在 50%~150% 之间缩放
shear	        0.0	    2.0	    随机错切角度	  模拟透视变形。小球变形后仍能被识别
perspective	  0.0	    0.0005	随机透视变换	  轻微模拟视角变化，不宜过大
erasing	      0.0	    0.5	    随机擦除概率	  模拟遮挡。小球被桌子/书本挡住时，模型仍能识别露出部分
hsv_h	        0.015	  0.02	  色调抖动	      模拟不同颜色的光照
hsv_s	        0.7	    0.8	    饱和度抖动	    模拟不同色彩饱和度
hsv_v	        0.4	    0.5	    明度抖动	      模拟不同亮度环境
fliplr	      0.5	    0.5	    水平翻转概率	  小球的对称性好，翻转不影响

调参原则：
a.一次只改一个变量,同时改多个参数无法判断哪个有效.
b.数据增强由弱到强,先跑通基础训练，再逐步加增强,避免一次性开太大.
c.学习率是最重要的超参数,过大学不精，过小学不完,Nano 模型建议 0.005~0.01.
d.早停不一定好，有时模型后期还会有小幅提升,可关闭早停跑满 epochs.
e.close_mosaic 很重要,训练后期关闭 mosaic,让模型适应真实尺度.
f.数据质量 > 参数调优,标注质量差、数据量太少,怎么调参都补不回来.
